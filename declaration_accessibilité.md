# Déclaration d'accessibilité

**Lucas Jouin Freelance** s'engage à rendre son portfolio accessible, conformément au Référentiel Général d'Amélioration de l'Accessibilité (**RGAA 4.1.2**).

Cette déclaration d'accessibilité s'applique au site **portfolio de Lucas Jouin** (`https://lulufreedesign.github.io/portfolio_lucasjouinfreelance/`).

## État de conformité

Le site est **totalement conforme** au RGAA 4.1.2 : **100 % des critères applicables sont respectés** (auto-évaluation).

> Il s'agit d'une auto-évaluation outillée (voir méthode ci-dessous), et non d'un audit réalisé par un organisme tiers agréé.

## Résultats des tests

L'auto-évaluation menée révèle que **56 critères sur 56 applicables sont conformes**, soit **100 %**.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (aucune dépendance ; améliorations progressives)

## Agents utilisateurs et technologies d'assistance utilisés pour vérifier l'accessibilité

- Chromium (moteur Blink) + axe-core 4.12
- Validation du code : html-validate
- Vérifications manuelles : navigation clavier, structure des landmarks, langue des passages, reflow à 320 px, contrastes.

## Environnement et méthode

Auto-évaluation combinant :
- **tests automatisés** rejouables (`a11y/rgaa-audit.mjs`, axe-core via Playwright) sur les 4 pages du site ;
- **revue experte manuelle** du code pour les critères non automatisables ;
- rapport détaillé : `a11y/RGAA-rapport.md`.
